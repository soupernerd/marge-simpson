# Preferences

> User's stated preferences for how things should be done. Explicit statements, not inferred.

## Entry Format
```markdown
### [PR-###] Short description #tag1 #tag2
- **Stated:** YYYY-MM-DD
- **Strength:** Weak / Moderate / Strong
- **Preference:** What the user prefers
- **Exceptions:** When this doesn't apply
- **Quote:** Direct quote if available
- **Related:** P-### or D-### entries
```
