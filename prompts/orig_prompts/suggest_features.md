# Feature Proposal Prompt

Read the AGENTS.md file in this folder and follow it.

**MODE:** PLANNING ONLY — no code changes, no patches, no execution.

---

## Goal

Propose 3-8 new features for this project.

---

## Requirements

- Rank by **end-user UX / value** (primary)
- These are proposals only — nothing approved yet
- Make reasonable assumptions, state them briefly

---

## For Each Feature

```
### [Rank]. Feature Name
**What:** 1-2 sentence description
**Who/Why:** Who benefits, why it matters
**Risk:** Biggest blocker or uncertainty
**Success:** How to validate it worked
```

---

## Output

1. Ranked feature list (highest value → lowest)
2. Top pick summary (3-4 lines)
3. Any assumptions made

Create: `./system/tracking/recommended_features.md`
