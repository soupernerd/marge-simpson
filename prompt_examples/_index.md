# Prompt Examples Index

> Copy-paste templates for common prompting scenarios. Pick one based on your intent.

## Quick Reference

| Intent | File | Description |
|--------|------|-------------|
| Bug fix, new feature, existing issue | [features_and_issues.md](features_and_issues.md) | Work items with tracking |
| General instructions | [misc_instruction.md](misc_instruction.md) | Custom instructions template |
| Mixed request (questions + work) | [multiple_prompts.md](multiple_prompts.md) | Combines multiple intent types |
| Questions or confirmations | [questions_confirmations.md](questions_confirmations.md) | Ask questions, confirm fixes |
| Feature brainstorming | [suggest_features_marge.md](suggest_features_marge.md) | PLANNING ONLY mode proposal |
| Codebase audit | [system_audit.md](system_audit.md) | Full system review |
| Iterative loop audit | [loop_audit](loop_audit) | Loop until clean with min/max iterations |

## When to Use Each

- **Starting new work?** → `features_and_issues.md`
- **Just have questions?** → `questions_confirmations.md`
- **Want ideas without code changes?** → `suggest_features_marge.md`
- **Full codebase review?** → `system_audit.md`
- **Multiple things at once?** → `multiple_prompts.md`
- **Iterative deep audit?** → `loop_audit`
